Giới thiệu về dự án blog

24/04/2026 Công nghệ 5 phút đọc

Mình bắt đầu với vài trang HTML tĩnh: trang chủ, chi tiết bài, form đăng nhập đơn giản. Chưa cần framework, chỉ cần bố cục gọn là đủ cho vài tuần đầu.

Phần CSS gom vào reset và một file style chung. Header và footer lặp lại giữa các file cho tới khi muốn tách partial hoặc build bước sau.

Khi cần thêm tương tác - chẳng hạn menu mobile, gửi form - lúc đó mới nối script vào từng chỗ, tránh nhồi hết vào một file từ đầu.
